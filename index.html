Le-co

Minimalistic-ish modular UI library

git repo

Configuration

For grids and other responsive patters you can specify set of breakpoints they will be generated against. 

$breakpoints: (
    "m": "481px",
    "l": "767px"
);

Example setting above will produce grid structure with classes like c-m-f1 that will kick in at 481px of width and c-l-f1 be effective at 767px and up.

There's configuration options in config.scss.

Layout

Container

Main .container can be used as generic "row" practically anywhere.

If combined with .container--fixed it will get centered and it will apply max-width specs based on breakpoint specification.
